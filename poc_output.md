## 1. Problem Statement

1. **The Unified Facade (Developer-Aligned)**
   - Create explicit /swt:flow subcommands for every underlying script operation.
   - Use industry-standard terminology (Backlog, History, Pulse, Context).
2. **Structural Integration**
   - Bridge the architectural gap identified by Graphify where skills were isolated from the orchestrator.
3. **Power User Rituals**
   - Surface specialized rituals like `test-fail` (TDD), `bug` (friction reporting), and `link-dev` (ongoing dev setup).
Turn the `swt:flow` orchestrator into a complete, 1-to-1 "Command Center" that reflects a modern open-source development workflow.
